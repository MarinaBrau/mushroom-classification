# Mushroom Classification

Projeto de **Machine Learning** para classificacao de cogumelos como comestiveis ou venenosos.

## Sobre

Este projeto aplica algoritmos de classificacao para prever se um cogumelo e comestivel ou venenoso com base em suas caracteristicas fisicas (formato do chapeu, cor, odor, habitat, etc).
